# Architecture: Build Optimizations & Engine Tuning

## Overview
To achieve the performance mandate of competing with Chromium-based browsers, Bobzilla requires aggressive build-time optimizations. This document outlines the specific compiler flags and architectural targets that must be enforced via `mozconfig.bobzilla` during the compilation of the `mozilla-unified` Gecko engine.

## 1. Profile-Guided Optimization (PGO)
PGO requires a two-stage build. The first stage instruments the binary, and the second stage compiles the final binary using profiling data generated by running a standard workload (e.g., loading complex Javascript-heavy webpages).
*   **Flag:** `ac_add_options --enable-profile-use=cross`
*   **Target:** `SpiderMonkey JIT compiler` and `WebRender` compositing threads.
*   **Benefit:** Achieves up to a 10-15% performance uplift in JavaScript execution and page layout calculations.

## 2. Link-Time Optimization (LTO)
LTO allows the compiler to perform optimizations across object file boundaries, aggressively inline functions, and eliminate dead code throughout the entire 3GB codebase.
*   **Flag:** `ac_add_options --enable-lto=cross`
*   **Target:** Global compilation.
*   **Benefit:** Reduces binary size, significantly improves instruction cache coherency, and provides a 5-8% general performance increase.

## 3. Rust-Specific Optimizations
Firefox heavily relies on Rust (especially for the Stylo/WebRender CSS engine). We must ensure Rust is instructed to compile for the specific host architecture of the target machine rather than generic compatibility.
*   **Flag:** `export RUSTFLAGS="-C opt-level=3 -C target-cpu=native"`
*   **Target:** `cargo` invocations within the `mach` build system.

## 4. Disabling Debug & Telemetry Overheads
Mozilla builds contain numerous debug assertions and telemetry hooks by default. Stripping these at compile-time prevents the engine from branching into unused reporting logic.
*   **Flags:**
    *   `ac_add_options --disable-debug`
    *   `ac_add_options --disable-tests`
    *   `ac_add_options --disable-crashreporter`
    *   `ac_add_options --disable-updater`
