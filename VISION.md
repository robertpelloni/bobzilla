# Vision Document: bobzilla

## Ultimate Goal
**bobzilla** aims to be the browser that doesn't suck. It combines the privacy DNA of Firefox, the performance competitiveness of Chromium/V8, proper Widevine L1 DRM support for 4K streaming, and zero surveillance out-of-the-box.

## Core Philosophy
1. **Privacy by default:** No opt-out telemetry. It is disabled natively.
2. **Performance matters:** Focus on SpiderMonkey optimization and memory efficiency.
3. **DRM pragmatism:** Widevine L1 must be supported to ensure functional media consumption.
4. **Ad Blocking Superiority:** Manifest V2 will be preserved forever to support tools like uBlock Origin natively.
